- PostgreSQL: 14.8

- AWS

## カリキュラム外の技術
- React

## 実行手順
1. `git clone git@github.com:monnichi1209/yaruzou.git`
2. `cd yaruzou`
3. `bundle install`
4. `rails db:create && rails db:migrate`
5. `rails s`
